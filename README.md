## ⚙️ Descripción

**PetSociety** es una plantilla base desarrollada en Django que proporciona una configuración inicial lista para usar en proyectos web.

Su objetivo es evitar comenzar desde cero, entregando una estructura funcional con herramientas ya integradas, permitiendo enfocarse directamente en el desarrollo de funcionalidades.

Aunque está ambientado en una veterinaria, el repositorio está diseñado como una **base reutilizable para cualquier tipo de sistema web**.

---

## 🎯 Objetivo del proyecto

👉 Proveer una base sólida y escalable para proyectos Django.

Incluye configuraciones y herramientas comúnmente utilizadas en desarrollo real, reduciendo tiempos de setup y facilitando la continuidad del desarrollo.

## ⚙️ Instalación

1. Clonar el repositorio:

```bash id="jz4s9d"
git clone https://github.com/KhadaKayn/PetSociety.git
cd PetSociety
```

2. Crear entorno virtual:

```bash id="0hfd6j"
python -m venv venv
```

3. Activar entorno:
# Windows
```bash id="o3f7qv"

venv\Scripts\activate
```
# Linux / Mac
```bash id="o3f7qv"

source venv/bin/activate
```

4. Instalar dependencias:

```bash id="g7n2vk"
pip install -r requirements.txt
```

5. Ejecutar migraciones:

```bash id="h7d2sl"
python manage.py migrate
```

---

## ▶️ Ejecución

